# Unity_iOS_MovieCam
Unity-iOSで外カメで動画を撮影して，カメラロールに保存する方法の備忘録



## Environment
- macOS Catalina 10.15 beta
- Unity2019.1.12f1
- XCode 11.1
- iOS 13.1.2
